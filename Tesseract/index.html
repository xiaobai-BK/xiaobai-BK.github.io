java解析页面验证码安装Tesseract

1.安装Tesseract

Tesseract项目最初由惠普实验室支持，1996年被移植到Windows上，1998年进行了C++化。在2005年Tesseract由惠普公司宣布开源。2006年到现在，都由Google公司开发。

目前，Tesseract的源代码放在GitHub上。

  • 进入下载页面，可以看到有各种.exe文件的下载列表，这里可以选择下载4.0版本。

  • 下载到本地后双击，会出现以下安装界面

  • 一路默认点击，如果需要其他语言，在这个页面勾选Additional language data(download)选项来安装OCR识别支持的语言包（用不着不用点）

  • 安装路径默认C:\Program Files (x86)\Tesseract-OCR
    到这个页面我一般是勾选不需要创建快捷方式的，因为也不会主动点击去用

    OK 这个时候咱们就已经安装好了 现在需要配置一下本地环境变量，然后就可以用了

    2.Windows下 Tesseract-OCR 的环境变量配置

  • 打开环境变量配置

  • 找到用户变量和系统变量 两个[Path]项点开，都添加刚才拷贝的Tesseract 的安装路径（由于刚才是默认安装，默认路径就是【C:\Program Files (x86)\Tesseract-OCR】）

  • 在[系统变量]中，新建一个变量名称为：TESSDATA_PREFIX
    值为：刚才路径加上 \tessdata

  • OK 安装配置完成 现在开始测试一下

    3.Tesseract-OCR 的文字识别测试

  • 咱们找个图片进行测试，最好图片放在一个比较好找的目录，我就直接放在了D盘了，路径为D:\test.jpg 这样的

  • 打开 cmd 进入该图片的文件夹
    使用 cd 目录名 进入目录

  • 输入： tesseract test.jpg demo –l eng
