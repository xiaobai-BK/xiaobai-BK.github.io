学习XML数据库（Marklogic）实战（1）

1.下载Marklogic Server

在非商业的情况下，Marklogic Server数据库是免费的，因为咱们是先学习Marklogic Server，所以先下载windows装在本机，易于操作，配合开发工具(IDEA)

目前，下载Marklogic Server需要注册登录MarkLogic社区
Marklogic Server下载地址.

  • 点击下载会弹出框让你先进行登录他的MarkLogic社区平台（sign up注册）

OK 这个时候咱们就已经下载好了，不是很难，但是国内下载很慢，建议梯子

2.Windows下安装Marklogic Server

  • Marklogic Server的安装就比较简单了，它不想mysql等数据库要配置东西，只需要点击软件安装，选择路径就OK
  • 我就不默认安装了，个人推荐选择一个盘符指定安装

开始菜单启动Admin marklogic server,和普通的win软件没什么异同。弹出的启动窗口启动完成后，自动关闭，说明安装已经完毕了。

3.配置Marklogic Server

  • 打开服务，找到MarkLogic，启动此服务
  • 在浏览器中输入localhost:8001,回车，能看到初始化配置界面，由于我们属于测试阶段，还用不到集群，可以直接忽略，点击skip。
  • 接下来，配置Admin用户名和密码。之后，在每次登陆的时候，网页都将提示我们输入次用户名和密码，请牢记(设置账号admin 密码11111–>好记)
  • 设置过后，就来到了后台管理页面了，以后没事会常来的，所以账号密码简单点

    控制台与激活

  • 在浏览器输入http://localhost:8000/qconsole/
    输入之前的账号密码，就来到了控制台页面，在这里，你可以测试你的demo
  • 在admin后台管理页面有一行说No license key has been entered，说我们还没有许可证，所以我们去申请一个许可证
    点击Get Deverloper License, 输入一些个人信息，同意Marklogic条约，即可免费申请到一个license key，当然它只能要求你在本机上使用，而且不能做商用。
  • 点击ok，server会自动重启，自动刷新Admin后台管理页面，发现未许可的提示已经没有了。现在你应该可以尽情地享用Marklogic了，按照官网的介绍，这个license只有6个月时间，之后需要再次申请。申请到了许可。
